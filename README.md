# What is FlutterTencentIm ？
FlutterTencentIM是基于[腾讯云即时通讯](https://cloud.tencent.com/product/im)服务进行的Flutter封装，为Flutter提供即时通讯服务.

# Sdk Version

| 平台    | SDK版本 |
|:--------|:-------|
| Android | 5.1.50 |
| IOS     | 5.1.50 |
